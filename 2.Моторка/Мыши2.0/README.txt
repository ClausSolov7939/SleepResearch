Обновление относительно первой версии "Мышей".

При первом запуске программы по умолчанию будут выставлены:
- fullscreen 0 == оконный режим с разрешением 1024х768
- inverse 0 == управление без инверсии
- round 20 == кол-во раундов 20
- possible 200 == размер квадратной зоны запрета управления 200x200 пикселей
- freq 0.25 == частота записи координат с каждого раунда в секундах
- radius_multiplier 1.5 == множитель радиуса попадания в нору относительно радиуса норы
Также эти настройки будут сохранены в файл config.txt, которые 
можно будет отредактировать под свои нужды.

После проигрывания всех раундов будут созданы:
- "config.txt" - конфигурация программы
- "main_log.txt" - тот же самый "log.txt" из прошлой версии
- папка "log_img", где будут сохранены картинки траекторий
- папка "log_txt", где будут сохранены тектовые файлы координат траекторий
В картинках теперь показана зона запрета управления

P.S.: Если файл config.txt будет заполнен не до конца программа будет 
      использовать и записанные настройки, и брать
      по умолчанию, те которые недописанные в файле.
      Рекомендуется в таких случаях (!только) удалять этот файл, 
      чтобы программа создала новый
